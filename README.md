### EXAM_DevOps
Проект представляет собой полный воспроизводимый DevOps-контур, включающий:
- **ML-пайплайн** для обработки и анализа датасета _Iris_:
    
    - проверка качества данных (Deepchecks)
        
    - анализ дрейфа (EvidentlyAI)
        
    - обучение модели RandomForest
        
    - логирование метрик и артефактов в MLflow
        
    - экспорт финальной модели
        
    
- **API-сервис (FastAPI)** для инференса модели:
    
    - эндпоинт /predict для получения предсказаний
        
    - /healthcheck для мониторинга
        
    - CI/CD выкладывает модель внутрь контейнера
        
    - деплой в **Yandex Cloud Serverless Containers**
        
    
- **Нагрузочное тестирование (Locust)** для оценки производительности API.

В проекте реализованы:

- контейнеризация (Docker), документирование архитектурных решений (ADR), автоматизация ML-пайплайна и сборки Docker-образа через GitHub Actions, а также подготовка к деплою через Terraform.
